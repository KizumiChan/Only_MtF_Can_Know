# Only_RLE_Can_Know
Only_RLE_Can_Know — Things Only Real-Life Experience Can Teach

## 如何贡献

欢迎任何人参与完善这份指南。你不需要是编程高手，会写文字就可以。

### 你不需要会 Git？也可以用网页版直接改

1. 打开你想修改的文章（在 `docs/` 文件夹里）
2. 点击右上角的 ✏️ 铅笔图标
3. 直接修改文字（Markdown 格式）
4. 在页面底部写清楚你改了啥，点击 **Propose changes**
5. GitHub 会自动帮你生成一个 Pull Request，我或其他维护者审核后就会合并

### 如果你想贡献一篇完整的文章

1. Fork 这个仓库
2. 在 `docs/` 文件夹下新建一个 `.md` 文件
3. 用 Markdown 写你的经验（可以附上配图）
4. 提交 Pull Request
